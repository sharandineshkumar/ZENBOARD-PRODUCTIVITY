**ZenBoard : Clean & Customizable Task Manager**

Robust **CRUD** functionality for Notes and Tasks persistent local storage  and real-time state management.

A fast and intuitive tool designed to streamline daily organization. ZenBoard provides a distraction free environment for **creating, managing, and archiving notes and tasks**.  Restoration enabled archive  and a polished user interface featuring custom theming Dark Mode and a fully responsive layout for seamless use
